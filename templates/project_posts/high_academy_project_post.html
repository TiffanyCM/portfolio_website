High Academy

High Academy is my senior project for Wilmington University. Currently, High Academy features Algebra I and Trigonometry courses. Each course consists of seven lessons that students can learn from and three quizzes they can complete.

Overview

This project uses Flask; a Python web framework used for developing websites and web applications. Since this was my first time using Flask, there was a noticeable learning curve. Through this project, I gained a deeper understanding of how web design involves not only displaying information on an HTML page but also creating visually engaging and user-friendly experiences.

After developing 46 HTML, CSS, and Python files, I was able to successfully complete the project. In the future, I hope to continue creating creative and user-friendly web applications using Flask.
